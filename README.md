# Поднять проект

- `git clone https://github.com/PM-YandexPracticum/SkillSwap_47_1.git` 
- `npm i`
- `npm run dev`

##  Команды

- `npm run dev` - запуск локального сервера разработки.
- `npm run build` - сборка проекта для продакшена.
- `npm run lint` - комплексная проверка (ESLint, Stylelint, Prettier).
- `npm run lint:fix` - аавтоматическое исправление ошибок линтеров и форматирование всех файлов.
- `npm run test` - запуск юнитов (Jest).
- `npm run test:coverage` - запуск тестов с генерацией отчета о покрытии (требование ≥ 70%).
