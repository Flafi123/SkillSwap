# Команды

- `npm run dev` - запуск локального сервера разработки.
- `npm run build` - сборка проекта для продакшена.
- `npm run lint` - комплексная проверка линтерами.
- `npm run lint:fix` - автоматическое исправление ошибок линтеров.
- `npm run test` — запуск юнитов (Jest).
- `npm run test:coverage` — запуск тестов с генерацией отчета о покрытии (требование ≥ 70%).
